### 🎮 AI Game Recommender
> *An AI-powered game recommendation engine built for my AIML course project*

A full-stack web app that takes your **PC specs** and **preferred genres** as input and uses the **Anthropic Claude API** to generate personalized game recommendations — complete with a dark neon/arcade-inspired UI.

**Stack:** `FastAPI` · `Python` · `Anthropic API` · `HTML/CSS/JS`

```
📁 game-recommender/
├── backend.py        # FastAPI server + Claude API integration
├── index.html        # Dark neon frontend with genre & spec dropdowns
└── requirements.txt  # Minimal dependencies
```
